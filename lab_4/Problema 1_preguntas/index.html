Problema 1

Preguntas

1. ¿Qué diferencias y semejanzas hay entre Java y JavaScript?

R:

  • Diferencias:
    • Java es un lenguaje de programación orientado a objetos y compilado, mientras que JavaScript es un lenguaje de scripting interpretado y basado en prototipos.
    • Java se usa principalmente en aplicaciones del lado del servidor y aplicaciones móviles, mientras que JavaScript se usa principalmente en desarrollo web del lado del cliente.
  • Semejanzas:
    • Ambos lenguajes comparten una sintaxis similar basada en C.
    • Ambos se pueden usar para el desarrollo web, aunque en diferentes contextos.

2. ¿Qué métodos tiene el objeto Date?

R:

  • getDate(): Devuelve el día del mes (1-31).
  • getDay(): Devuelve el día de la semana (0-6).
  • getFullYear(): Devuelve el año completo (4 dígitos).
  • getMonth(): Devuelve el mes (0-11).
  • getHours(): Devuelve la hora (0-23).

3. ¿Qué métodos tienen los arreglos?

R:

  • push(): Añade uno o más elementos al final de un array y devuelve la nueva longitud del array.
  • pop(): Elimina el último elemento de un array y lo devuelve.
  • shift(): Elimina el primer elemento de un array y lo devuelve.
  • unshift(): Añade uno o más elementos al inicio de un array y devuelve la nueva longitud del array.
  • forEach(): Ejecuta una función proporcionada una vez para cada elemento de un array.

4. ¿Cómo se declara una variable con alcance local dentro de una función?

R: Para declarar una variable con alcance local dentro de una función, se utiliza la palabra clave var, let o const dentro de la función. Por ejemplo: 

function miFuncion() {
    let variableLocal = "Local";
}

5. ¿Qué implicaciones tiene utilizar variables globales dentro de funciones?

R:

  • Las variables globales pueden ser modificadas por cualquier parte del código, lo que puede llevar a errores difíciles de depurar.
  • El uso excesivo de variables globales puede afectar el rendimiento de la aplicación debido al mayor tiempo de búsqueda de variables en el ámbito global.
  • Pueden causar conflictos de nombres si diferentes partes del código utilizan las mismas variables globales para propósitos diferentes.
